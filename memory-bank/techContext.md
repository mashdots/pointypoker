# Tech Context

## Technologies

- **Framework:** React 18
- **Language:** TypeScript
- **Styling:** CSS Modules and styled-components.
- **State Management:** Zustand
- **Backend:** Firebase (Firestore, Authentication, Hosting)
- **Build Tool:** Vite
- **Package Manager:** Yarn

## Development Setup

1. Clone the repository.
2. Install dependencies with `yarn install`.
3. Set up Firebase credentials in a `.env` file.
4. Run the development server with `yarn dev`.

## Technical Constraints

- **Real-time Communication:** The application must support real-time updates, which is why Firebase is a core dependency.
- **Browser Compatibility:** The application should be compatible with the latest versions of modern browsers (Chrome, Firefox, Safari, Edge).
